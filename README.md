# Install Libs
```
$ npm install
```
# How to run 
```
$ npm run dev # run for developer
$ npm run build # run build for production
```

# Deploy 
```
$ npm run build
```
- Copy folder dist to server
```
$ node dist/index.js
```
- Use can using pm2 start for background
